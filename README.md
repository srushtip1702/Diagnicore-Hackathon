DIAGNICORE
One AI Engine. Multiple Domains.
DiagniCore is an offline-first AI-powered diagnostic assistant that helps users analyze problems across Healthcare, Agriculture, Manufacturing, and Business domains using a unified AI decision engine.

# Problem Statement
Build intelligent systems, neural networks, language classifiers, and predictive tools to scale automated decision making.

# Solution Overview
DiagniCore is a multi-domain diagnostic platform that provides preliminary AI-powered analysis across various sectors.

The system:
* Collects domain-specific user inputs.
* Performs AI-based risk analysis.
* Generates temporary recommendations.
* Provides explainable AI outputs.
* Stores previous records locally.
* Identifies historical patterns and similar cases.
* Works in offline-first environments.

Supported domains:
* Healthcare
* Agriculture
* Manufacturing
* Business

# Installation and Setup Instructions
## Frontend

```bash
cd frontend
npm install
npm start
```

Application runs at:

```text
http://localhost:3000
```

---

## Backend

```bash
cd backend

pip install fastapi
pip install uvicorn

uvicorn main:app --reload
```

Backend runs at:

```text
http://127.0.0.1:8000
```

---

## Run Complete Application

1. Start FastAPI backend.
2. Start React frontend.
3. Open localhost:3000.
4. Begin diagnosis.
